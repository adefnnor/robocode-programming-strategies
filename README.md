# Programming and Strategies using Robocode

This repository contains the code developed for my final project for my Bachelor’s Degree in Physics, focused on programming and strategy development using Robocode.

Robocode is an open-source programming game in which users develop autonomous combat robots that compete against each other in a simulated battlefield. Each robot is programmed to sense its environment, make strategic decisions, and react dynamically during combat.

The goal of this project is to explore:

* Algorithmic strategies for robot combat

* Decision-making in adversarial environments

* Optimization of movement, targeting, and defense behaviors

* Performance evaluation of different programming approaches

The repository includes the robot implementations, supporting code, and experiments carried out as part of the project.
